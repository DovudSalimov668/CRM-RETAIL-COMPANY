Staff/Admin Documentation

Complete guide for managing the Retail CRM system

Getting Started

  1. Login: Navigate to /staff/login/ or use the "Staff Login" option from the home page.
  2. Dashboard: After login, you'll see the main dashboard with key metrics and statistics.
  3. Navigation: Use the sidebar menu to access all CRM features.

Customer Management

Customer List

  • View all customers with search and filter options
  • Filter by status (Lead, Prospect, Active, Inactive, VIP)
  • Filter by source (Website, Referral, Social Media, etc.)
  • Search by name, email, phone, or company
  • Export customers to CSV format

Create Customer

  • Add new customers with complete profile information
  • Upload profile pictures (stored in media/profiles/)
  • Set customer type (Individual, Business, VIP)
  • Assign status and source

Customer Details

  • View complete customer profile and history
  • See all orders, interactions, and deals
  • Edit customer information
  • Track customer analytics and RFM scores

Order Management

Order List

  • View all orders with status filters
  • Search orders by number, customer, or date
  • Filter by payment status and order status

Create Order

  • Manually create orders for customers
  • Add multiple products with quantities
  • Set shipping address and notes
  • Calculate totals automatically

Update Orders

  • Change order status (Pending, Processing, Shipped, Delivered, Cancelled)
  • Update payment status
  • Modify order items and quantities
  • Add notes and tracking information

Product Management

Product List

  • View all products with thumbnails
  • See stock levels and pricing
  • Filter by category and status

Create Product

  • Add products with images (upload or URL)
  • Set SKU, name, description, and category
  • Configure pricing and stock quantity
  • Upload images stored in media/products/

Product Details

  • Update product information
  • Manage stock levels
  • Change pricing and status
  • Update product images

Reports & Analytics

Dashboard Analytics

  • View total customers, orders, and revenue
  • Monthly revenue charts
  • Customer status distribution
  • Recent activity and orders

Reports Page

  • Sales reports and statistics
  • Top customers and products
  • Revenue trends and analysis
  • Deal pipeline overview

Task Management

  • Create and assign tasks to team members
  • Set due dates and priorities
  • Track task completion status
  • Link tasks to customers or deals

Deal Management

  • Create and track sales deals
  • Set deal stages (Prospecting, Qualification, Proposal, Negotiation, Closed Won/Lost)
  • Assign deal values and probabilities
  • Link deals to customers

Quote Management

  • Generate quotes for customers
  • Add products and services to quotes
  • Set expiration dates
  • Convert quotes to orders

Advanced Features

Customer Segmentation

  • Create dynamic customer segments
  • Set criteria for automatic segment assignment
  • View segment statistics

Marketing Campaigns

  • Create and manage marketing campaigns
  • Target specific customer segments
  • Track campaign performance (sent, opened, clicked)

Support Tickets

  • Manage customer support tickets
  • Set priority levels and status
  • Track response times
  • Communicate with customers via ticket messages

Loyalty Programs

  • View customer loyalty points and tiers
  • Track point transactions
  • Manage loyalty program settings

Automation Workflows

  • Create automated workflows
  • Set triggers and actions
  • Track workflow executions

RFM Analysis

  • Analyze customer Recency, Frequency, and Monetary value
  • View RFM segments
  • Identify high-value customers

Cart Abandonment

  • View abandoned shopping carts
  • Track recovery rates
  • Send reminder emails to customers

Technical Notes

  • Media Files: Profile pictures are stored in media/profiles/ and product images in media/products/
  • CSV Export: Customer data can be exported from the Customer List page
  • Permissions: Staff pages require authentication and redirect to /staff/login/ if not logged in
  • Database: The system uses PostgreSQL for data storage
  • Image Uploads: Both file uploads and image URLs are supported for products

Quick Tips

  • Use the search and filter options to quickly find customers, orders, or products
  • Check the dashboard regularly for key metrics and recent activity
  • Export customer data regularly for backup purposes
  • Keep product stock levels updated to avoid overselling
  • Use tasks to track follow-ups and important deadlines
  • Monitor the deal pipeline to forecast revenue
